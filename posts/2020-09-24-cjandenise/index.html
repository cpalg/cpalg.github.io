CJ và Denise - LQDOJ.

2020-09-23
LQDOJ Đồ thị , DFS , Chu trình Euler , Đường đi Euler , Lý thuyết

Link đề bài:

  • http://lqdoj.edu.vn/problem/cjanddenise.

Hướng dẫn:

  • Bài toán trên là định nghĩa về chu trình Euler: Chu trình Euler là chu trình mà xuất phát từ một đỉnh bất kì, đi qua tất cả các cạnh của đồ thị và trở về đỉnh ban đầu.

  • Một đồ thị là một chu trình Euler khi và chỉ khi nó liên thông và tất cả các đỉnh của nó đều bậc chẵn.

  • Để xác định đường đi trong chu trình Euler, ta áp dụng thuật toán sau:

    • Xuất phát từ một đỉnh u đang xét, với mỗi cạnh (u, v) ta sẽ xóa bỏ cạnh v khỏi đồ thị, và duyệt DFS qua v, cuối cùng thêm u vào đường đi.

  • Bạn có thể tham khảo link cuối trang để tìm hiểu thêm về chu trình và đường đi euler.

Code mẫu:

#include <bits/stdc++.h>
#define fi first
#define se second
#define ii pair<int, int>
using namespace std;

const int MAX = 4e5+6;

int n, m, In[MAX];
bool vs[MAX];
vector<ii> g[MAX];
vector<int> EulerPath;

void DFS(int u){
    while (g[u].size()){
        int v = g[u].back().fi;
        int id = g[u].back().se;
        g[u].pop_back();
        if (vs[id]) continue;
        vs[id] = true;
        DFS(v);
    }
    EulerPath.push_back(u);
}

int main(){
    ios_base::sync_with_stdio(0); cin.tie(0); cout.tie(0);
    //freopen("input.txt","r",stdin);
    cin >> n >> m;
    for ( int i=1; i<=m; i++){
        int u , v;
        cin >> u >> v;
        g[u].push_back(ii(v,i));
        g[v].push_back(ii(u,i));
        In[u]++; In[v]++;
    }
    bool bad = false;
    for(int i=1; i<=n; i++) {
        if (In[i]%2) bad = true;
    }

    DFS(1);

    // Do thi la chu trinh euler khi va chi khi
    // bac cua tat ca cac dinh la chan va do thi lien thong
    if (bad || (EulerPath.size() != m+1)){
        cout << "IMPOSSIBLE";
        return 0;
    }

    for(int i=0; i< EulerPath.size(); i++) cout << EulerPath[i] << " ";
    return 0;
}
