Cách đếm số hoán vị mà không có phần tử nào nằm đúng vị trí. - Lý thuyết

2020-09-22
Toán Tổ hợp, xác Xuất , Lý thuyết

Bài toán:

Cho môt dãy n phần tử gồm các hoán vị của dãy từ 1 tới n, đếm số hoán vị mà trong các phần tử, không có phần tử nào nằm đúng vị trí.

Cách làm:

  • Gọi C[i] là số hoán vị gồm i phần tử mà không có phần tử đúng vị trí.
  • Ta có: C[i] = (i-1) * (C[i-1] + C[i-2]) . Với C[1] = 0, C[2] = 1

Chứng minh:

Giả sử ta đã tạo được dãy hoán vị với i-1 phần tử, giờ ta sẽ đặt phần tử thứ i vào tập có tất cả n-1 cách thêm phần tử i:

  1. Phần tử i được đặt vào một vị trí j nào đó và phần tử j lại được đặt vào vị trí i, có (n-2) cách chọn.
  2. Phần tử i được đặt vào một ví trí j mà j đặt khác vị trí của i, có (n-1) cách chọn.

Có tất cả C[i] = (i-1) * (C[i-1] + C[i-2]) cách chọn.

Áp dụng:

  • Ta có thể áp dụng để làm bài tập này: Lều thi.

Hướng dẫn:

  • Để đếm số cách chọn có đúng k đội may mắn, ta sẽ chọn ra trước k đội, còn lại n-k đội, ta sẽ áp dụng bài toán trên để tìm số các hoán vị của k-1 phần tử mà không có phần tử nào nằm đúng vị trí. Kết quả sẽ là C(n,k) * D[n-k]

Code mẫu:

#include <bits/stdc++.h>
#define READFILE freopen("input.txt", "r", stdin)
#define ll long long
using namespace std;

const int MAX= 1e5+6;
const ll BASE = 1e9+7;

ll n, k, gt[MAX], gtNg[MAX], D[MAX];

ll C(int k, int n){
    return ((gt[n] * gtNg[k]) % BASE * gtNg[n-k]) % BASE;
}

ll Pow(ll a, ll b){
    if (b==1) return a;
    ll x = Pow(a,b/2);
    x = (x*x) % BASE;
    if (b&1) x=(x*a)% BASE;
    return x;
}

int main()
{
    #ifndef ONLINE_JUDGE
        READFILE;
    #endif

 	cin >> n >> k;
 	gt[0] = 1;
 	for(int i=1; i<=n; i++) {
        gt[i] = (gt[i-1]*i) % BASE;
        gtNg[i] = Pow(gt[i], BASE-2);
    }
    D[0] =1; D[1] = 0; D[2] = 1;
    for(int i=3; i<=n; i++){
        D[i] = (i-1) * (D[i-1] + D[i-2]) % BASE;
    }
    ll res = (D[n-k] * C(k,n)) % BASE ;
    cout << res;
    return 0;
}
